# ServiceNow Ticketing System Lab (Incident Management & SLA Simulation)

## Objective

Simulated a real-world IT help desk environment using ServiceNow. Created and managed incident tickets, configured users and groups, handled ticket assignment and resolution workflows, and implemented SLA tracking to monitor response times.

### Skills Learned

- Hands-on experience with ServiceNow Incident Management and ticketing workflows
- Understanding of ticket lifecycle management (New → In Progress → Resolved → Closed)
- Ability to triage and prioritize incidents using impact and urgency
- Experience configuring users, roles, and support groups
- Knowledge of ticket assignment and escalation processes
- Practical use of work notes and resolution documentation
- Understanding of SLA (Service Level Agreement) configuration and tracking
- Exposure to IT service management (ITSM) principles and best practices

### Tools Used

- ServiceNow (Personal Developer Instance) – Used to simulate a real-world IT help desk environment and manage incident tickets
- ServiceNow Incident Management Module – Created, tracked, and resolved tickets through the full lifecycle
- ServiceNow User Administration – Configured users, roles, and groups to replicate IT support structure
- ServiceNow SLA Module – Implemented Service Level Agreements to monitor response times
